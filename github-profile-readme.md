## Step-by-step

1. Go to https://github.com/new
2. Repository name: **`mouxbax`** (must match your username exactly)

4. Check "Add a README file"
5. Click "Create repository"
6. GitHub will tell you "You found a secret! Your README is now visible on your profile."
7. Click the pencil icon to edit `README.md`
8. Replace the default content with the markdown below
